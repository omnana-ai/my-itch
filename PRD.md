# Product Requirements Document (PRD) - My-Itch

## 1. Introduction
My-Itch is a web platform inspired by itch.io, designed for independent game developers to host, sell, and share their creations. It focuses on simplicity, community, and discoverability.

## 2. Core Features
### 2.1 User Accounts
- **Developers**: Can upload games, set prices, customize game pages, and view analytics.
- **Gamers**: Can browse, buy/download games, rate/review, and follow developers.
- **Admins**: Content moderation, user management.

### 2.2 Game Hosting
- Support for HTML5 (web-playable) and downloadables (Win, Mac, Linux).
- File versioning.
- Custom game page styling (CSS/Theme editor).

### 2.3 Discovery & Social
- Algorithmic feeds (New & Popular, Top Rated).
- Tags and Categories.
- User Collections (Lists of games).
- Comments and Devlogs.

### 2.4 Payments (Phase 2)
- "Pay what you want" model.
- Integration with Stripe/PayPal.

## 3. User Flow
1. **Landing Page**: Featured games, search bar, login/signup.
2. **Dashboard (Dev)**: Manage projects, analytics, payout settings.
3. **Game Page**: Description, screenshots, download/play button, comments.
4. **Browse Page**: Filtering by genre, platform, tag.

## 4. Success Metrics
- Number of active developers.
- Daily active users (DAU).
- Game uploads per week.
